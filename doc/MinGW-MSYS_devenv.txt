MinGW and MSYS Soothsayer
~~~~~~~~~~~~~~~~~~~~~~~~~

Install MinGW
~~~~~~~~~~~~~
Download and install the current version of the following components:
 MinGW base tools
 g++ compiler


Install MSYS
~~~~~~~~~~~~
Install MSYS by repeatedly clicking Next and then answer Yes to continue with the post install script.


Install Python
~~~~~~~~~~~~~~
Install using Python Windows Installer. No special notes here, just click next.


Install wxPython
~~~~~~~~~~~~~~~~
Install using wxPython runtime installer matching the installed Python version.
Just click next, and make sure that site-packages directory the installer picks up is correct.


Install SWIG
~~~~~~~~~~~~
Install SWIG for Windows by simply unzipping it.


Build SQLite3
~~~~~~~~~~~~~
Get the SQLite3 amalgamation source distribution and unzip it (i.e. C:\SQLite3 )
Start MSYS and change into the SQLite3 directory:
  cd /c/SQLite3 )
Build the SQLite3 DLL, import library and definition file:
  gcc -shared -o sqlite3.dll sqlite3.c -Wl,--output-def,sqlite3.def,--out-implib,libsqlite3.a


Build soothsayer
~~~~~~~~~~~~~~~~
Start MSYS console
Extract and configure soothsayer, taking care of specifying where to find required binaries (Python and SWIG), headers, and libraries:

  PATH=/c/Python25:/c/SWIG:$PATH \
  CPPFLAGS="-I/c/SQLite3 -I/c/Python25/include" \
  LDFLAGS="-L/c/SQLite3" \
  ./configure --enable-python-binding --prefix=/c/soothsayer
